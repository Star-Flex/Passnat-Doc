### 无 GUI 操作

无 GUI 操作是指: 使用 CMD 或 PowerShell 启动 Frpc 程序。此方式应为 Frpc 默认使用方式。

#### 文字版

- 根据自己的系统平台，选择对应架构的二进制文件。本文以 amd64 版为例，如为 arm 平台请选择 arm64

  :::info

  - 提示: 如果操作系统的 Microsoft Defender 误报病毒，请将二进制文件加入白名单。[点我跳转微软文档操作](https://support.microsoft.com/zh-cn/windows/%E5%B0%86%E6%8E%92%E9%99%A4%E9%A1%B9%E6%B7%BB%E5%8A%A0%E5%88%B0-windows-%E5%AE%89%E5%85%A8%E4%B8%AD%E5%BF%83-811816c0-4dfd-af4a-47e4-c301afe13b26)。
  - 提示: 请注意下载来源，可以从本站下载也可以从 Frp 作者 Github 下载，其余渠道下载请注意甄别是否安全。
    :::

- 前往 PassNAT 控制台创建隧道，直接下载配置文件。
- 在存储 frpc 文件的目录下，直接打开 CMD 或 PowerShell ，确保运行路径为当前文件路径 输入

  :::tip

  - 小技巧1 :在文件管理器顶部直接输入 `cmd` 或 `powershell` 即可在目录路径下打开，使用 `tab` 按键补全文件即可。
  - 小技巧2 :小技巧1实在不会也可以直接 **拖拽文件** 进入 `cmd` 或 `powershell`，会自动补全文件路径，中间使用`-c`拼接即可。
    :::

```cmd title="注意: 当前为示例代码，请根据实际情况填写，不要直接复制本代码运行"
./frpc.exe -c frpc.toml
```
